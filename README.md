<h4>我们的目标：</h4>
<p>1.拓展插件，尽可能覆盖业务所用的常用组件</p>
<p>2.吸收jQuery，MV*框架的优点，开始新一轮的造轮子工程。</p>
<p>3.欢迎前后端的小伙伴提意见，测试，加入和贡献代码。</p>
<h4>关于代码管理的sw_define的使用，请查看这里https://github.com/whitekyo/sunivoweb-ui/blob/master/demo.html</h4>
