表格 - table

颜色 色块 色值 class名
蓝色 #0099ff text-blue, btn-primary
橙色 #eb6100 text-warning, btn-warning
紫色 #5637aa text-info, btn-info
红色 #e60012 text-error, btn-danger
绿色 #009900 text-success, btn-success

TIPS

.table为最基本的表格，只有横向边框，.table-bordered表示全边框，.table-condensed表示较为紧缩的表格。
表格默认有20px的margin-bottom。

表单 - form

input.email，邮箱自动补全
select.chosen
input.date-picker, data-date-start-date:起始时间
input.time-picker, data-show-seconds='true':显示秒
input.portSelector, data-cn='true/false':中/英文, data-region='foreign/domestic':国外/内数据
  

上传组件.file-group中包括：label.file-label(必需，入口), input:file(上传表单，必需), .file-return(如果为input则会将返回的name填充，如果为img则会将返回的url填充到src，其它类型则会将返回的info填充到text)

TIPS

.control-group默认有20px的margin-bottom，.control-label的默认宽度是130px，水平表单需要在form上加form-horizontal的class。

CSS HACK

CSS HACK BAR
/** CSS HACK **/
.hack{
    background: yellowgreen;   /** for all **/
    background: orange\9;      /** IE6+ **/
    background: #008000\0;     /** IE8+ **/
    *background: cyan;         /** IE7、IE6 **/
    _background: red;          /** IE6 **/
}

:root .hack{
    background: blue\9;        /** IE9+ **/
}

@media screen and (-webkit-min-device-pixel-ratio: 0) {
    .hack {background: deeppink;}  /** webkit **/
}
